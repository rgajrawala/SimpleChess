# SimpleChess
A simple Chess game made in C++.

## HowTo
+ [Get started with SFML.](http://www.sfml-dev.org/tutorials/2.3/#getting-started)
+ `git clone https://github.com/usandfriends/SimpleChess.git`
+ `cd bin`
+ `cmake .. && make`
+ `./SimpleChess`
